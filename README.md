# README
## Ruby version
Ruby '2.6.1'

## Rails Version
Rails 6.0.3.4

## System dependencies
Install webpacker with "run rails webpacker:install"

## Gems used
Used postgres as the database for Active Record
gem 'pg'

For authentication
gem 'devise'

## Database creation
rake db:create

## Database initialization
rake db:migrate

## Start application on local
rails s -p 3000
